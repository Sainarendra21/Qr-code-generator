# QR Code Generator

This is a simple web application that generates QR codes based on the user's input. Users can enter text and select a size for the generated QR code, and upon clicking the "generate" button, a QR code is displayed on the page.

## Features

- Generate QR codes of different sizes (100x100, 150x150, 200x200, 250x250, 300x300).
- Enter custom text to be encoded into the QR code.
- Interactive user interface.

## Getting Started

1. Clone the repository:

`gh repo clone Sainarendra21/Qr-code-generator`

2. Open the `index.html` file in a web browser.

## Usage

1. Open the `index.html` file in a web browser.
2. Enter the text you want to encode into the QR code in the textarea.
3. Select the desired size for the QR code from the dropdown menu.
4. Click the "Generate" button.
5. The generated QR code will be displayed on the page.

## Technologies Used

- HTML
- CSS (defined in `style.css`)
- JavaScript (defined in `script.js`)
- Google Chart API for generating QR codes.
